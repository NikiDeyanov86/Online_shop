# Online shop
Online shop

## Installation
### Linux
```bash
# install 
virtualenv env
source env/bin/activate
pip install -r requirements.txt
```
### Windows
```bash
# install  
python -m virtualenv .
.\scripts\activate
pip install -r requirements.txt
```

## Run
### Linux
```bash
# run
export FLASK_ENV=development
flask run
```
### Windows
```bash
# run
set FLASK_ENV=development
flask run
```
